# SEO Execution Checklist (Google-Safe)

This checklist covers remaining non-code execution items after technical SEO implementation.

## 1) Google Search Console Execution (Weekly)

1. Inspect these URLs in GSC:
- `/led-display/`
- `/km/led-display/`
- `/solutions/retail-digital-signage-cambodia/`
- `/solutions/hotel-display-solutions-cambodia/`
- `/products/catalog/led-display/`
- `/products/catalog/indoor-led-display/`
- `/products/catalog/outdoor-led-display/`
2. Confirm:
- `Page is indexed`
- `User-declared canonical = Google-selected canonical`
- `Alternate page with proper hreflang tag` is detected for EN/KM
3. Use `Request Indexing` only after major content/meta updates.
4. Track query deltas for:
- `led display cambodia`
- `led display price cambodia`
- `indoor led display cambodia`
- `outdoor led display cambodia`
- Khmer variants with `អេក្រង់ LED`

## 2) Core Web Vitals Monitoring (Weekly)

1. Check GSC `Core Web Vitals` report (mobile first).
2. Prioritize pages with poor `LCP` and `INP`.
3. Check PageSpeed Insights for `/led-display/` and top 3 catalog pages.
4. Keep a simple changelog before/after each performance change.

## 3) Local SEO (GBP + NAP) (Biweekly)

1. Google Business Profile:
- Primary category aligned with core service
- Service areas: Phnom Penh, Siem Reap, Sihanoukville
- Business description aligned with LED display installation + service
2. Ensure NAP consistency across:
- Website footer
- LocalBusiness schema
- Google Business Profile
- Local citations

## 4) Citations & Backlinks (Monthly)

1. Submit/verify local citations (high-quality only).
2. Publish at least 1 partner/project mention with backlink per month.
3. Focus anchor context:
- `LED display Cambodia`
- `LED installation Cambodia`
- branded anchors (`Mugnee Cambodia`)

## 5) Case Study Publishing (Monthly)

1. Publish 1-2 Cambodia project case studies:
- Project context
- Screen type + pixel pitch
- Installation timeline
- Before/after outcome
2. Link every case study to `/led-display/` and relevant product pages.

## 6) CTR Optimization Loop (Every 14 Days)

1. Pull GSC page+query report for:
- `/led-display/`
- top solution pages
- top catalog pages
2. Identify low CTR, high impression queries.
3. Update title/meta in small batches (no mass rewrite).
4. Re-measure after 14 days.
