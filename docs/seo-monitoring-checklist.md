# SEO Monitoring Checklist (Post Deploy)

## Goal
- Brand queries should land on `/`
- LED product-intent queries should land on `/led-display`

## Search Console Mapping
1. Open `Performance` report.
2. Filter last `28 days`.
3. Add page filter:
- `Page = https://mugneekh.com/`
- `Page = https://mugneekh.com/led-display/`
4. Compare query groups:
- Brand: `mugnee`, `mugnee cambodia`
- LED intent: `led display cambodia`, `led display price cambodia`, `indoor led display cambodia`, `outdoor led display cambodia`, `អេក្រង់ LED`

## Weekly Review (2-4 Weeks)
1. Check homepage for non-brand query growth.
2. Check `/led-display/` for LED-intent query growth.
3. If homepage ranks for LED queries, reduce LED-heavy anchors/text on homepage.
4. If `/led-display/` underperforms, strengthen internal links from homepage/footer and add fresh LED proof content.

## Success Signal
- `Brand CTR` on `/` increases.
- `LED CTR` and impressions on `/led-display/` increase.
- Query/page intent alignment becomes stable over 2-4 weeks.
